# Source Schema

Schema Version: v0.1

## Purpose

Represents evidence supporting archive records.

## Permanent ID

SRC-000001

## Required Fields

- Source ID
- Title
- Source Type
- Author
- Publisher
- Date
- URL or Location
- Access Date
- Language
- Copyright
- Confidence Level

## Source Types

- Official Website
- Newspaper
- Magazine
- Book
- Interview
- Concert Flyer
- Program
- Social Media
- Personal Archive
- Audio Recording
- Video
- Photograph
- Other

## Confidence Levels

- A : Official / Primary
- B : Reliable Secondary
- C : Community Verified
- D : Unverified

Every archive record must reference at least one Source.
