# Organization Schema

Schema Version: v0.1

## Purpose

Represents organizations related to jazz culture.

## Permanent ID

ORG-000001

## Required Fields

- Organization ID
- Official Name
- Alternate Name(s)
- Organization Type
- Country
- Region / City
- Founded
- Dissolved (if applicable)
- Website
- Notes

## Organization Types

- Venue Operator
- Festival
- Record Label
- University
- Jazz Club
- Music School
- Promoter
- Association
- Media
- Other

## Relationships

An Organization may operate Venues.

An Organization may organize Events.

An Organization may employ or represent Musicians.
