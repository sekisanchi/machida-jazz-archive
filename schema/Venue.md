# Venue Schema

Schema Version: v0.1

## Purpose

Represents a venue where jazz-related activities take place.

## Permanent ID

VEN-000001

## Required Fields

- Venue ID
- Official Name
- Alternate Name(s)
- Country
- Region / City
- Address
- Latitude
- Longitude
- Open Date
- Close Date (if applicable)
- Status (Active / Closed / Relocated)
- Owner / Operator
- Capacity
- Piano
- PA System
- Website
- Notes

## Relationships

A Venue may host many Events.

A Venue may be operated by an Organization.

A Venue may have multiple historical names.
