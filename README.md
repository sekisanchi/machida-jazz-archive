# Machida Jazz Archive

An open, collaborative archive for preserving the history and network of jazz culture, beginning with Machida and expanding to Japan and the international jazz community.

## Mission

Preserve the activities, relationships, venues, and history of jazz culture as an open knowledge base.

## Initial Scope

- INTO THE BLUE (Machida)
- Nica's (Machida)

## Data Policy

Every record should include:

- Source
- Confidence level
- Last verified date
- Permanent ID
